PilotOS Sync
Atajo de iOS — eCrews
El atajo lee tu horario mensual de eCrews directamente desde el DOM de la página y te lleva a PilotOS con los datos listos para importar. Sin contraseñas, sin APIs, sin timeout.
¿Cómo funciona?
Safari (eCrews) → Compartir → PilotOS Sync → el script lee las actividades del calendario → Safari abre PilotOS automáticamente → pulsas "Importar". Listo.
Instalación (una sola vez)
1
Copia el script
Pulsa el botón azul.
2
Atajos → nuevo atajo ( + )
App naranja → nuevo atajo.
3
Añadir acción → "JavaScript"
Busca "JavaScript" → "Ejecutar JavaScript en página web".
4
Pega el script
Toca dentro del bloque de código → Seleccionar todo → Pegar.
5
Nombre + Panel Compartir
Nombre: "PilotOS Sync". Pulsa ⓘ → activa "Mostrar en panel Compartir".
Uso diario
✓
Safari → eCrews → My Schedule → Compartir → PilotOS Sync
Safari abre PilotOS automáticamente. Pulsa "Importar" en el banner azul.
💡 Solo 1 acción en el Atajo. No necesitas "Get Contents of URL" ni tokens — el script se encarga de todo navegando directamente a PilotOS con los datos.
