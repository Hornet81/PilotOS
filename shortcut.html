PilotOS Sync
Atajo de iOS
Instala el Atajo de Siri una vez. Después: Safari → eCrews → Compartir → PilotOS Sync.
Instalación
1
Copia el script
Pulsa el botón azul.
2
Abre Atajos → +
App naranja → nuevo atajo.
3
Añadir acción → "JavaScript"
Busca "JavaScript" → "Ejecutar JavaScript en página web".
4
Pega el script
Dentro del código → Seleccionar todo → Pegar.
5
Nombre + Compartir
Nombre: "PilotOS Sync". Pulsa ⓘ → activa "Mostrar en panel Compartir".
✓
Uso
Safari → eCrews My Schedule → Compartir → PilotOS Sync → IMPORTAR.
💡 Solo se instala una vez.
