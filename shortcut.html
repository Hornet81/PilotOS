PilotOS Sync
Atajo de iOS — instalación
Instala el Atajo de Siri una vez.
Después: en eCrews → Compartir → PilotOS Sync → el roster se importa automáticamente.
Instalación (solo una vez)
1
Copia el script
Pulsa el botón azul de arriba.
2
Abre la app Atajos
Icono naranja. Pulsa + para crear un atajo nuevo.
3
Añade la acción JavaScript
Pulsa Añadir acción → busca "JavaScript" → selecciona "Ejecutar JavaScript en página web".
4
Pega el script copiado
Dentro del bloque de código → Seleccionar todo → borrar → Pegar.
5
Activa en panel Compartir
Nombre: "PilotOS Sync". Luego pulsa ⓘ abajo → activa "Mostrar en panel Compartir" → OK.
✓
Cómo usarlo
Safari → eCrews → My Schedule → botón Compartir → PilotOS Sync.
El panel se abre, descarga los datos y muestra IMPORTAR EN PILOTOS. Un tap y listo.
💡 Solo se instala una vez. Después funciona con un tap en cualquier página de eCrews.
