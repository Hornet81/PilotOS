PilotOS Sync
Atajo de iOS — eCrews
El atajo descarga tu horario mensual de eCrews usando el endpoint Export de AIMS y te lleva a PilotOS con los datos listos para importar. Sin contraseñas, sin APIs externas.
¿Cómo funciona?
Safari (eCrews) → My Schedule → Print → Compartir → PilotOS Sync → el script descarga el HTML del reporte via Export → Safari abre PilotOS → pulsas "Importar". Listo.
Instalación (una sola vez)
1
Copia el script
Pulsa el botón azul.
2
Atajos → nuevo atajo ( + )
App naranja → nuevo atajo.
3
Añadir acción → "JavaScript"
Busca "JavaScript" → "Ejecutar JavaScript en página web".
4
Pega el script
Toca dentro del bloque de código → Seleccionar todo → Pegar.
5
Nombre + Panel Compartir
Nombre: "PilotOS Sync". Pulsa ⓘ → activa "Mostrar en panel Compartir".
Uso diario
✓
Safari → eCrews → My Schedule → Print → Compartir → PilotOS Sync
Safari abre PilotOS automáticamente. Pulsa "Importar" en el banner azul.
💡 Requisito: el popup de impresión de eCrews debe estar abierto antes de ejecutar el Atajo. Abre My Schedule → Print, espera que cargue el visor, luego Compartir → PilotOS Sync.
